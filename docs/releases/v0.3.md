# v0.3 — Live Captions + Time-Based Segmentation

**Tag:** `v0.3`
**Branch:** `feat/live-captions-ui` (merged to `main`)
**Date:** 2026-06-17

## Summary

GTK live captions window for Linux, pluggable segmenter architecture with time-based mode as the new default, and faster feedback via `--interval`. VAD mode remains available as an option.

## Added

### GTK Live Captions Window
Always-on-top, transparent, undecorated GTK3 captions window.

```bash
python -m src captions
python -m src captions --interval 5
python -m src captions --mode vad
```

Key files:
- `src/ui/gtk_window.py` — `CaptionWindow` class (transparent, dark, draggable)
- `src/__main__.py` — `captions_command()` entry point

### Time-Based Segmentation (Default)
- No VAD dependency needed — segments are a fixed duration (default 15s)
- Faster start: no wait for speech to begin
- `--interval` flag for custom duration
- `TimeSegmenter` in `src/transcription/segmenter.py`

### Pluggable Segmenter Architecture
- `TimeSegmenter` / `VadSegmenter` share the same `add_audio(chunk)` interface
- `LiveStream` uses `self.segmenter` — swap modes without rewriting pipeline
- New mode flag: `--mode time` (default) or `--mode vad`

### New CLI Flags

| Flag | Applies to | Default | Description |
|---|---|---|---|
| `--mode` | listen, captions | `time` | `time` or `vad` |
| `--interval` | listen, captions | `15.0` | Seconds per time segment |
| `--silence_duration` | listen, captions | `0.64` | Silence threshold (VAD mode) |

## Changed

### `src/transcription/stream.py`
- Before: hardcoded silero-vad pipeline, `SpeechBuffer` class
- After: `self.segmenter` (pluggable), created in `__init__` based on mode
- Bug fix: time mode no longer double-accumulates audio — now passes `chunk` (not `frame_buffer`) to segmenter

### `src/transcription/segmenter.py` (new file)
Extracted from `stream.py`:
- `TimeSegmenter` — fixed-interval accumulation
- `VadSegmenter` — speech-bounded (migrated from old `SpeechBuffer`)
- Shared constants: `FRAME_SIZE = 512`, `PRE_BUFFER_FRAMES = 17`

### `src/config.py`
Added settings:
- `mode: str = "time"`
- `interval_sec: float = 15.0`
- `silence_duration_sec: float = 0.64`

### `src/__main__.py`
- `captions_command()` — new function for `python -m src captions`
- `listen_command()` — added `mode`, `interval`, `silence_duration` params
- `fire.Fire` dict updated with `captions` entry
